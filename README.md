# Resume-EZ
1.0 of a Resume Tracker / recommender which can help navigate application processes. The Program uses csvs to store applications from the new application screen where you can fill out status and company position the comp a job offers aswell as any extra notes someone might have on a job like remote in person maybe the sector youd specifically be working in. The whole idea came from being sick of the overextrapulated job finding process in the world of tech instead of a hundreds of rows long excel sheet this is an interactable nice looking helpful consolidating place to look at insights. The goal of this project is to make the process of all application processes for people navigating the modern world of work culture with lots being expected with things like Handshake, LinkedIn, Indeed and juggling that with school and trying to stand out from the rest it can be overwhelming for most Resume-EZ is here to fix that.
# Tools
The Project will consist of JS, SQL, CSS, Bootstrap and the API WeatherStack. Desktop designed and meant to be a low impact application for computers so it can be used on phones and laptops.
