# Resume-EZ
1.0 of a Resume Tracker / recommender which can help navigate application processes.
## 
The goal of this project is to make the process of all application processes for people navigating the modern world of work culture with lots being expected with things like Handshake, LinkedIn, Indeed and juggling that with school and trying to stand out from the rest it can be overwhelming for most. Resume-EZ is here to fix that
# Tools
The Project will consist of JS, SQL, CSS, Bootstrap and API usage yet to be decided
